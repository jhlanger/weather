# Weather Forecast App

## Purpose
This is a weather dashboard that takes a city input from the user and displays a daily and 5 day forecast. The forecast will show weather conditions in the form of an image, temperature, humidity and windspeed. In the Daily forecast the UV index is displayed and is color coded depending on severity.
